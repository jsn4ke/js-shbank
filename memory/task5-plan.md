# Task 5: 数据解析器实现计划

## 目标
实现 Parser 类，将 API 返回的原始数据解析为 ProductModel 对象，并进行字段映射转换。

## 小功能点清单

### 5.1 创建 Parser 基础结构
- 创建 `src/scraper/parser.py`
- 实现 Parser 类
- 添加字段映射辅助方法

### 5.2 实现标签解析方法
- 实现 `parse_labels(labels_str)` - 解析 @!@ 分隔的标签字符串
- 处理空字符串和 None 值

### 5.3 实现字段映射转换
- 风险等级数字转描述
  - 0=""（无风险）
  - 1="R1极低风险"
  - 2="R2低风险"
  - 3="R3中风险"
  - 4="R4中高风险"
  - 5="R5高风险"
- 币种代码转换 (CNY="人民币", USD="美元")
- 布尔值转换 (0="否", 1="是")
- 数值格式化（起售金额添加单位）

### 5.4 实现产品记录解析
- 实现 `parse_product(record)` - 解析单条产品记录
- 字段名从驼峰转换为下划线
- 应用字段映射转换
- 处理可选字段和空值

### 5.5 实现响应解析
- 实现 `parse_api_response(response)` - 解析完整 API 响应
- 提取 records 列表
- 批量解析产品记录

### 5.6 编写测试 (TDD - 先写测试)
- 创建 `tests/test_parser.py`
- 测试标签解析
- 测试字段映射转换
- 测试产品记录解析
- 测试完整响应解析
- 测试空值和特殊值处理
